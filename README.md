1. Hitung Diskon
2. Hitung Faktorial
3. Hitung Luas
4. Kalkulator Sederhana
5. Konversi Mata Uang
6. Konversi Suhu
7. Membalik String
8. Pendataan Mahasiswa
9. Rental PS
10. Tebak Angka
11. Game Petani-Jagung-Domba-Serigala
